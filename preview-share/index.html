v3 架构 vs v1 架构(5 个关键改进)

  • Modal 双层:预览不离开 share 页,点 ▶ 弹全屏 Modal(视频/图片/PDF/音频/文本),不新建路由
  • 5 密度档:L0(1) / L1(2-5) / L2(6-15) / L3(16+) 自动适配,不"1 个嫌空 200 个嫌挤"
  • 双主题:light / dark / auto 三态,Modal 也适配(原有 share 页未支持)
  • 跨文件操作:L1+ 出现"打包下载"主 CTA + 列表项"▶ 下一项"手工切换
  • 5 种 viewer:SDP 视频 / Image / PDF.js / 音频 / 文本,不只视频
  • 3 断点响应式:≥1024 桌面 / 640-1023 平板 / <640 手机,同套代码
  • ThemeToggle:右上角 Sun/Moon/Monitor 循环切换(原本漏的,已加)
  • 16+ 网格:L3 状态显示缩略图墙 + 左侧筛选侧栏 + 批量选择
  • 免费预览:无付费墙(预留之后补),Modal 入口直接进入
  • 共享 Modal:多文件共享同一 Modal,"X / N"指示 + 上一项/下一项
