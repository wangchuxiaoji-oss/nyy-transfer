## 命名

| 项 | 值 |
| --- | --- |
| 中文名 | 拿呀呀 |
| 拼音 | ná yā yā（缩写 nyy） |
| 域名 | nyy.app |
| 英文别名 | Nayaya（仅文档 / SEO 用，UI 不展示） |
| Slogan | 想传文件？拿呀呀。 |
| English Tagline | Just grab it. |

## 调性

- 关键词：温暖、轻盈、邻家、可靠、不油腻
- 受众感觉：像便利店里随手能拿到的小物件 — 不昂贵，但好用
- 反面：技术冷感（"区块链分布式"）、卡通幼稚（"萌系迪士尼"）、高冷设计感（"Apple 白"）

## Logo 用法

- 主 Logo：`logo.svg`（卡片 / 落地页头部）
- 文字 Logo：`logo-wordmark.svg`（仅汉字 + 域名，导航条用）
- 头像 Logo：`logo-mark.svg`（圆形吉祥物头像，favicon / PWA icon / 分享缩略）
- 最小尺寸：吉祥物头 ≥ 32px，主 Logo ≥ 96px 宽
- 留白：四周至少留出"拿"字字符高度的 50%
- 禁用：拉伸 / 反色 / 加描边 / 用作图案重复

## 吉祥物（拿宝）

形象：圆头、橘色短毛、双手前伸做"接住文件"的动作；表情友好不卖萌；可演化为多种姿态（举手、招手、捧礼盒、思考）。

后续物料：404 页、空列表、分享成功页、移动端 splash。
